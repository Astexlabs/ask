# contributing to ask by astexlabs

ask is a fork of [zev](https://github.com/dtnewman/zev). we welcome contributions that keep the project simple and maintainable.

no AI Slop please - the project is maintained by one person ([@ableez](https://github.com/ableez)) and we can’t review endless lines of code. if you want a feature but dont code no problem, open an issue, if you really cant wait, vibecode your way privately and enjoy ;).

## how to contribute

- **small changes** (bug fixes, typos, docs): open a pr.
- **larger changes** (refactors, new features): open an issue first to discuss.

keep prs focused on one change. for features, a short screen recording in the pr helps review.

## code style

follow existing style in the repo. run `ruff check` and `ruff format` before submitting.

## questions

open an issue or contact the maintainers.

thank you for contributing.
